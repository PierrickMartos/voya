# Voya

AI-powered travel planning. Describe your trip, get a personalized day-by-day itinerary.

**Stack:** React 19, TypeScript, Vite, Tailwind CSS v4, Vitest  
**AI:** LLMLayer API (GPT-4o with live web search)  
**Hosting:** Render

## Setup

```bash
cp .env.example .env   # add VITE_LLMLAYER_API_KEY
npm install
npm run dev
```

## Commands

```bash
npm run dev      # Dev server
npm run build    # Production build
npm run test     # Run tests
npm run lint     # Lint
```
